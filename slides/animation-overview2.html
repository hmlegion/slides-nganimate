Non animation based CSS we need to be comfortable with.

  • Position (relative and absolute)
  • Transforms (translate, scale, skew, rotate)
  • Opacity
  • Rule of thumb: Just about anything with a number property can be animated
  • Gradients are a big exception here (although you can get around that)
  • Things that have key-words... not so much.
